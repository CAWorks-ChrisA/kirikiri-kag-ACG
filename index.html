Welcome to the Kirikiri Project by ACG.

This project is just a small fun project, where me, and a few of my friends, decide to just make a small visual novel. So, we gathered a programmer(me), a story-writer/game designer, 2 artists, and a UI designer, and we begin our journey.

About this project

This project's final product is a game, specifically, a visual novel, which tells the story of one great hero, fighting for his beloved. Where we work hard to deliver a visually immersive experience to the readers.

System

This game uses Kirikiri Z Engine. I would like to give credit to the creator of this engine. This engine is used to create lots of great visual novels, and it is maybe the only open-source visual novel dedicated engine out there. Thank you so much for giving programmers around the world to use such a great engine.
